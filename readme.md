
## USE

Put your images into subfolders in the /source folder and use:

```
<img src="/image.php?w=600&h=360">
```

## Params

- **w** - width
- **h** - height
- **f** - format (resize or crop). Crop as default value
- **t** - type of image (folders from /sources)
- **ph** - show size of image as text
